In this lab, I worked with a color image and converted it from BGR format to RGB format. I then applied several arithmetic operations to study their effects on image brightness and contrast.

### Implemented Tasks
The following tasks were implemented in Python:

- Load and display the original image  
- Increase brightness using the `cv2.add` function  
- Decrease brightness using `cv2.subtract`  
- Increase contrast using `cv2.multiply`  
- Decrease intensity using `cv2.divide`
